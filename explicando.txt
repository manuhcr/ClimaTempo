Diferença entre clima.py (script Python) e app.py (Flask web app)
1. O que é o clima.py?
Script simples em Python

Funciona no terminal (linha de comando)

Pede ao usuário para digitar o nome da cidade usando input()

Consulta a API do clima (OpenWeatherMap)

Mostra o resultado no terminal com print()

Exemplo:

python
Copiar
Editar
cidade = input("Digite a cidade: ")
dados = obter_clima(cidade)
mostrar_clima(dados)
2. O que é o app.py com Flask?
Aplicação web em Python

Roda um servidor local que você acessa pelo navegador

Recebe o nome da cidade via formulário HTML (página web)

Consulta a API do clima

Retorna uma página HTML com o resultado formatado (com CSS e layout)

Exemplo de rota no Flask:

python
Copiar
Editar
from flask import Flask, render_template, request

@app.route("/", methods=["GET", "POST"])
def index():
    if request.method == "POST":
        cidade = request.form.get("cidade")
        dados = obter_clima(cidade)
        # enviar dados para template
    return render_template("index.html", clima=dados)
3. Diferenças principais
Aspecto	clima.py	app.py (Flask)
Como recebe a cidade	input() no terminal	Formulário HTML via navegador
Como mostra resultado	print() no terminal	Renderiza página HTML com dados
Uso	Local, só você no terminal	Qualquer pessoa pelo navegador
Interface	Texto simples	Página web com layout e CSS
Código	Funções + execução linear	Rotas, templates e servidor web

@app.route("/", methods=["GET", "POST"])
def index():
    clima = None
    erro = None
    if request.method == "POST":
        cidade = request.form.get("cidade")
        if cidade:
            dados = obter_clima(cidade)
            if dados:
                clima = {
                    "nome": dados.get("name"),
                    "pais": dados["sys"].get("country"),
                    "descricao": dados["weather"][0].get("description", "").capitalize(),
                    "temp": dados["main"].get("temp"),
                    "sensacao": dados["main"].get("feels_like"),
                    "umidade": dados["main"].get("humidity"),
                    "vento": dados["wind"].get("speed"),
                }
            else:
                erro = "Cidade não encontrada."
        else:
            erro = "Digite uma cidade válida."
    return render_template("index.html", clima=clima, erro=erro)

if __name__ == "__main__":
    app.run(debug=True)
